코어 시스템 플러그인의 역할 

설명 : 코어 시스템 플러그인은 게임의 기본적인 요소들을 충복하기 위해 만들어진 플러그인으로 다음과 같은 역할을 수행한다.


1. 게임 사이클 시스템 
>> 게임을 3가지 요소로 나누자면 메인 화면 / 로비 화면 / 인 게임 화면으로 나눌 수 있다. 따라서 게임 사이클 시스템은 이 3가지 요소가 적절하게
작동할 수 있도록 전체적인 사이클을 매니저 형식으로 가진 채 이벤트 패턴으로 이를 진행하도록 만드는 기반을 제공한다.

2. 메인 타이틀 시스템 
>> 게임의 메인 화면은 어느 게임이나 마찬가지로 존재해야 하는 화면이다. 따라서 이를 시템으로 나누어 어느 게임이 되었든 모듈화된 방식으로 뺏다
넣었다 하여 적절한 원하는 메인 타이틀을 만들 수 있도록 만드는 시스템이다.

3. 로비 시스템
>> 어떤 게임에는 로비가 존재할 수 있고, 어떤 게임에는 로비가 존재하지 않을 수 있다. 그러므로 옵션적으로 넣거나 뺄 수 있도록 만든 시스템으로
로비 시스템이 필요한 경우, 이 시스템을 레고처럼 갔다 넣기만 하면 바로 작동되도록 만든 시스템이다.

4. 세이브 로드 시스템
>> 어떤 게임이든 세이브 로드가 없는 게임은 없다. 따라서 여러 종류의 세이브 로드 구조에 대응할 수 있는 기초적인 구조를 제공하는 세이브 로드
시스템은 어떤 게임이든 곧바로 커스터마이징이 가능하도록 만든 시스템이다.
